# System Requirements

AlterNET UI Version 1.0.1 requires .NET 8.0 or newer.

The following operating systems are supported:

- Windows 11, Windows 10 (version 1607+), and Windows Server 2012 R2 or later.
- macOS 10.15 or newer.
- Linux (different distributions).

See [this page](https://github.com/dotnet/core/blob/main/release-notes/8.0/supported-os.md) for detailed information.

**Microsoft Visual Studio 2022 and 2026** are supported by the **AlterNET UI Visual Studio Extension**.

AlterNET UI also supports a development workflow with **Visual Studio Code** and `dotnet` command line tools.

AlterNET UI works on the MAUI .NET 10.0 platform. There is a ControlView and its various descendants, which allow you 
to use AlterNET UI controls inside MAUI applications.
