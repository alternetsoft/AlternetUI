## What's New in 0.9.759-0.9.802 

This beta adds support for .NET 10, MAUI 10, and Visual Studio 2026. We continue improving our framework by adding new controls and fixing bugs.

Main Improvements:

- Added support for .NET 10 and Visual Studio 2026.
- Updated to use MAUI 10.
- Made more compatible with the different Linux distributions.
- Fixed UIXML previewer issues.
- Added OpenGL rendering support for UserControl.
- Added dark mode support for Windows.
- Bug fixes and optimizations
